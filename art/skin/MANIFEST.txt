# Underfoot art manifest — player skins (view.js SKINS, by skin id)
# Format: transparent PNG, square. Recommended 256x256. Filenames are exact and
#   case-sensitive and MUST equal the skin's `id` in view.js SKINS. A file with no
#   matching SKINS entry is an orphan (never renders); a SKINS entry with no file
#   silently falls back to the emoji below.
# Each line: <filename>  — <Name> skin (<how it unlocks>). Hero when equipped
#   (battle hero, Workshop character, skin picker, player stat panel). Fallback <emoji>.
# Unlock source of truth: view.js SKINS (unlock type + cost/mult/hint); condition
#   skins are flipped by Engine.evaluateSkinUnlocks or an unlockSkin event.
